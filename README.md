AI-powered content generation system for 11Solutions, automating LinkedIn content creation from client calls.

## Tech Stack

- **n8n** - Workflow automation
- **Fathom** - Call recording & transcription
- **Notion** - Content database
- **OpenAI GPT-4** - Content generation
- **Perplexity** - Research & trend analysis
